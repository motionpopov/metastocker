# 🚩 ПРАВИЛО №1: ВЕРСИИ
После ЛЮБЫХ правок в коде ты ОБЯЗАН:
1. Увеличить версию в `index.html` (тег `<span class="badge">v...</span>`).
2. Отразить изменения в текущем списке правок в этом файле.

## История версий:
- **v2.5**: Исправлено CSV-экранирование: значения с запятыми, включая списки тегов, снова остаются в одной колонке в экспортах Adobe, Envato и Shutterstock.
- **v2.4**: Добавлен ручной переключатель светлой и тёмной темы в шапку рабочего экрана; выбор сохраняется в браузере, а цвета, модальные окна и зона импорта адаптированы для тёмного режима.
- **v2.3**: Поле Batch comments переименовано в понятное описание batch: контекст добавляется к каждому промпту, а пример больше не содержит запрета на бренды.
- **v2.2**: Для обязательных тегов добавлен выбор позиции: в начале, в конце или случайно среди сгенерированных тегов. Режим фиксируется для запущенного batch и применяется к таблице и CSV-экспортам.
- **v2.1**: Поле Access Token переименовано в OpenAI API key; добавлены понятная подсказка и отдельное иллюстрированное руководство для новичков по регистрации в OpenAI API, настройке биллинга, созданию и безопасному использованию ключа; статья добавлена в блог.
- **v2.0**: Новый дизайн принят как основной. Исправлены сбои Web Worker, удаление и повторный импорт файлов, двойные и зависшие API-запросы, валидация превью и точного числа тегов, частичные экспорты, CSV-экранирование и счётчики ошибок; усилены безопасность ключа, доступность модальных окон и адаптивность, исправлены фильтры и навигация блога, Tailwind CDN заменён статической production-сборкой.
- **v1.57**: Обновлен дизайн рабочего экрана и адаптивная верстка, улучшены подсказки и клавиатурная навигация; исправлены ошибки Threads, редактора системного промпта, подсказок моделей и сохранения настроек, а также небезопасный вывод имен файлов и логов.
- **v1.55**: Добавлен бейджик подсчета точного количества сгенерированных тегов в таблице напротив каждого файла.
- **v1.54**: Добавлен счетчик желаемого количества тегов (с подсказкой Adobe), ползунок потоков заменен на числовое поле ввода для удобства, промпты стали динамическими.
- **v1.53**: Скрыт переключатель "Flex Mode" из интерфейса. Теперь режим удешевления (Flex) всегда включен под капотом, а формула стоимости пересчитана на 50% дешевле.
- **v1.52**: Интеграция автоматического кэширования промптов (Cached Input), пересчет стоимости токенов и вывод их в интерфейсе.
- **v1.51**: Добавлена поддержка .MOV, расчет стоимости токенов GPT-5.4, удаление/перегенерация отдельных файлов.
